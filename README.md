# Sai World City Panvel - Luxury Landing Page

A modern, responsive luxury real estate landing page for **Sai World City Panvel**.

**Built with React + Tailwind CSS**, with Framer Motion animations for polished interactions and scroll-based entrances.

## Features

- Responsive layouts for mobile, tablet, and desktop
- Sticky navigation with smooth section scrolling
- Premium dark theme with consistent yellow accents
- Hero section with layered gradient overlay and animated CTA panel
- Scroll-triggered section entrances using Framer Motion
- Animated pricing table, amenities cards, gallery, testimonials, and footer
- Contact form and repeated call-to-action buttons
- Expanded footer with quick links, contact details, and social icons

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- React Icons

## Getting Started

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Highlights

- `Navbar.jsx`: Sticky navigation with backdrop blur and animated links
- `Hero.jsx`: Luxury hero with stronger overlay contrast and animated CTAs
- `FloorPlans.jsx`: Pricing table with staggered row animation
- `Amenities.jsx`: Animated amenities grid with card lift hover states

- `Testimonials.jsx`: Spacious animated testimonial cards
- `Footer.jsx`: Project details, quick links, contact details, and social icons
